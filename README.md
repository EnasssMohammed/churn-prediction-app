# churn-prediction-app

##  Project Overview
This project predicts customer churn using machine learning and provides an interactive Streamlit app.

##  Business Goal
Help businesses identify customers at risk of leaving and take proactive actions.

##  Model
- Model: Logistic Regression
- Features: Age, Gender, Tenure, Monthly Charges

##  Tech Stack
- Python
- Scikit-learn
- Streamlit
- NumPy
- Joblib
